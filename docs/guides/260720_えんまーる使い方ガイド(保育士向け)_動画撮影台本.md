### カット3:サインアップ・登録(00:40–01:10)

- **画面/操作(実機収録・テストアカウントで実演)**:
  1. トップページを表示 → ヘッダーの「新規登録」をタップ
  2. 登録区分選択画面で「保育士・保育補助希望の方」カードをタップ
  3. 利用規約同意画面までスクロールし、「利用規約に同意します」にチェック
  4. 「登録を完了する」をタップ → 完了、マイページへ遷移
  5. 画面下部のLINE友だち追加案内が表示される箇所で1〜2秒静止
- **テロップ**: 「①新規登録 → ②役割を選択 → ③規約に同意 → ④登録を完了する」
- **ナレーション**: 「まずは新規登録です。トップページの『新規登録』ボタンから進み、『保育士・保育補助希望の方』を選択します。利用規約に同意して『登録を完了する』を押せば、登録は完了です。登録後は、マッチングのお知らせなどを受け取れるように、LINE公式アカウントの友だち追加もお願いしています。」
- **収録メモ**: 一連の操作は1テイクの通し撮りを推奨。画面にメールアドレス等が映る場合は編集でぼかし処理する。

  2. 表示名・希望エリア・保有資格・保育経験・得意分野・自己紹介の入力欄を上から順にスクロールしながら映す(すでに入力済みのダミーデータを見せる形でよい)
  3. 「保存する」をタップ → 保存完了のスナックバー表示
- **テロップ**: 「本名・ブランク期間・職務経歴はマッチング成立後の保育園にのみ公開」
- **ナレーション**: 「次に、マイページから『プロフィール編集』を開きましょう。表示名や希望エリア、お持ちの資格、保育経験、得意分野、自己紹介などを入力します。本名やブランク期間など一部の情報は、マッチングが成立した保育園にだけ公開される仕組みなので、安心して詳しく書いてください。入力できたら『保存する』を押します。」

  3. 「アップロード」ボタンをタップ→ダミーのサンプルファイルを選択→アップロード完了、ステータスが「確認中」になる様子を映す
- **テロップ**: 「対応形式:JPEG・PNG・WebP・PDF(各10MBまで)」
- **ナレーション**: 「応募の前に、必要な書類の提出をお願いします。健康診断書と履歴書は必須で、保育士証や検便結果は募集内容によって必要になります。『アップロード』ボタンからスマホの写真やPDFを選ぶだけで提出でき、対応形式はJPEG・PNG・WebP・PDF、10メガバイトまでです。運営が内容を確認し、『認証済み』になると応募できるようになります。」

### カット6:保育園を探して応募する(02:15–02:50)

- **収録環境**: マッチング成立・チャット・業務完了報告・評価(カット7〜11)は、実際のアプリの状態遷移に依存するため、本番収録前に必要な状態のテストデータを段階ごとに準備しておくこと(一発撮りでは繋がらない)。
